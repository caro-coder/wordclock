# Functions

### Set color
You can set a foreground and background color on your clock.

#### API example
`http://[YOUR_WEMOS_IP]/?mode=time&fg=#ffffff&bg=#000000`

### Set time offset
You can set a time offset to specify your timezone.

#### API example
`http://[YOUR_WEMOS_IP]/?mode=time&tz=3`
